# 📊 ScoutNOVO - Sistema de Análise e Coleta de Dados

Sistema completo para coleta, processamento e análise de dados, desenvolvido com backend em Python e deploy em nuvem, focado em automação e organização de informações.

---

## 🧠 Sobre o projeto

O **ScoutNOVO** foi desenvolvido com o objetivo de automatizar a coleta e centralização de dados, permitindo análise eficiente e tomada de decisão baseada em informações organizadas.

O projeto simula um ambiente real de aplicação, envolvendo backend estruturado, banco de dados em nuvem e integração com frontend.

---

## 👨‍💻 Minha atuação

Atuei no desenvolvimento completo do sistema, incluindo:

- Desenvolvimento do backend com Flask  
- Estruturação de rotas e lógica da aplicação  
- Integração com banco de dados (Supabase)  
- Deploy do backend na nuvem (Render)  
- Organização e tratamento dos dados  

---

## ⚙️ Tecnologias utilizadas

- Python  
- Flask  
- Supabase (Banco de dados)  
- Render (Deploy backend)  
- JavaScript / HTML / CSS  
- Git e GitHub  

---

## 🔄 Funcionalidades

- Coleta de dados automatizada  
- Armazenamento em banco de dados na nuvem  
- API para manipulação de dados  
- Integração entre frontend e backend  
- Estrutura escalável para novas funcionalidades  

---

## 🌐 Deploy

O backend da aplicação foi publicado utilizando o **Render**, permitindo acesso remoto e simulação de ambiente real de produção.
